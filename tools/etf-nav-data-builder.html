ETF NAV Data Builder

Owner tool — keep local. Generates etf-nav-data.js for the Nifty ETF calculator, using the NAV history of India's oldest Nifty ETF, Nifty BeES (Nippon India ETF Nifty 50 BeES, launched Dec 2001).

How to get the data (free):
1. Find the scheme code: open https://api.mfapi.in/mf/search?q=nifty%20bees in your browser and note the schemeCode for "Nippon India ETF Nifty 50 BeES" (often 101206).
2. Open https://api.mfapi.in/mf/<schemeCode> — you'll see JSON with a data array of {date, nav}.
3. Copy that whole JSON (or just the data array) and paste it below. The tool also accepts CSV rows like dd-mm-yyyy,nav.
